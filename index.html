FAQ - 服务器成员帮助指南

欢迎各位小伙伴加入我们的《我的世界》生电或者是机械动力服务器！此文档适用于疑难解答。

适用于超绝生电服务器（1.21.4）和王老帅与lizhiMC服务器（机械动力1.19.2）这两个服务器的帮助。

一、连接服务器

游戏链接获取

  1. 官方渠道：若你尚未下载《我的世界》游戏，可前往官方网站（minecraft.net）进行下载。官网提供了针对不同系统（如 Windows、Mac、Linux 以及移动设备的 iOS 和 Android 系统）的游戏版本，根据自己的设备类型选择对应的下载选项即可。
  2. 游戏平台：在一些游戏平台也能获取《我的世界》，例如微软商店（适用于 Windows 系统），打开微软商店后，在搜索栏输入 “Minecraft”，找到游戏后点击下载安装。对于主机玩家，如 Xbox、PlayStation 等，可在对应的主机游戏商店中搜索并下载《我的世界》。
  3. 离线玩家请下载第三方启动器，如pcl2等等进行下载

基岩版连接

  1. 打开游戏后，点击主界面的 “游玩” 选项。
  2. 加入公开服务器：在 “服务器” 分页浏览在线服务器列表，找到我们的服务器后点击 “加入服务器”。若没找到，确认服务器是否正常开启，或联系管理员询问。
  3. 加入特定服务器：向管理员获取服务器地址，在服务器分页底部点击 “新增服务器”，填好名称和地址后点击 “储存”，后续就能从列表中快速加入。

Java 版连接

  1. 启动游戏，在主菜单点击 “多人游戏”。
  2. 连接线上服务器：获取服务器地址后，点击 “直接连线” 输入地址连接；或点击 “新增服务器” 输入信息保存，方便下次加入。
  3. 版本检查：连接失败若提示版本不匹配，在游戏启动器切换到与服务器一致的版本。有些启动器可在设置中添加特定版本核心，下载切换后再尝试连接。

二、游戏配置

基岩版配置

  1. 图形设置：进入游戏后，点击主菜单中的 “设置” 选项，选择 “视频” 标签。在这里，你可以调整分辨率，降低分辨率能提升游戏流畅度，例如从 1920x1080 调整为 1366x768。同时，将 “图形” 品质设置为 “流畅”，关闭 “光影” 效果，减少设备图形处理压力。
  2. 音效设置：在 “设置” 的 “声音” 标签下，可根据喜好调整背景音乐、音效等音量大小。如果想减少游戏对系统资源的占用，也可以适当降低音效质量或关闭部分音效。
  3. 控制设置：在 “设置” 的 “控制” 标签中，自定义操作方式，如调整移动、跳跃、攻击等按键，让操作更符合自己的习惯，提高游戏操控体验。

Java 版配置

  1. 启动器设置：打开 Java 版启动器，点击 “启动选项”。在 “Java 设置” 部分，可根据电脑配置调整内存分配，例如电脑内存充足，可将 “最大内存” 设置为 4096M（4GB），以提升游戏运行性能；若电脑内存有限，适当降低内存分配，避免游戏卡顿。同时，确保 Java 版本与游戏版本兼容，可在 “Java 路径” 中选择正确的 Java 安装目录。
  2. 游戏内设置：进入游戏后，在主菜单点击 “选项”。在 “视频设置” 中，同样可调整分辨率、图形品质（如将 “渲染距离” 调小，从 16 降低到 8）、关闭 “平滑光照” 等，优化游戏画面显示效果，提升流畅度。在 “音频设置” 中，调节声音相关参数，在 “控制设置” 中自定义按键操作。

三、被 Ban IP 解决办法

  1. 联系管理员申诉：若发现自己的 IP 被服务器封禁，首先要保持冷静。通过我们的服务器官方交流群、论坛私信或管理员提供的其他联系方式，礼貌地向管理员说明情况。详细阐述自己是否存在误操作或可能导致封禁的原因，请求管理员核实并解除封禁。例如，你可以这样表述：“管理员您好，我发现自己的 IP 无法连接服务器，想询问是否被封禁。我一直遵守服务器规则，不知是否存在误封情况，希望您能帮忙核实并解除封禁，非常感谢！”
  2. 检查自身网络问题：在等待管理员回复期间，检查自身网络是否存在异常。确认网络连接是否稳定，可尝试访问其他网站或在线服务，看是否能正常加载。检查防火墙或安全软件设置，确保没有阻止《我的世界》游戏程序访问网络。如果使用路由器，查看路由器的访问控制列表，看是否存在限制自身 IP 访问的规则。
  3. 特殊情况处理：若管理员确认封禁原因合理且无法解除，而你确定后续会遵守规则，可尝试与管理员沟通解封期限。若封禁时间较长，且你急切想继续游戏，在了解服务器规则允许的前提下，可考虑使用合法的网络代理服务（如部分地区允许的 VPN）来更换 IP 地址，但务必注意使用此类服务需遵守相关法律法规以及服务器规则，避免造成更严重的违规。

四、游戏内常见问题解决

无法正常游戏

游戏卡顿

  1. 自身设备原因：降低游戏图形设置，比如调低分辨率、关闭光影特效，减轻设备运行压力。
  2. 服务器问题：若多人都卡顿，可能是服务器性能不足。及时在游戏内聊天框或官方交流群告知管理员，管理员会优化服务器配置，如增加内存、调整参数。

物品丢失

  1. 先回忆是否因死亡掉落且未及时拾取，若是，回到死亡地点寻找。
  2. 若不是，可能是插件冲突或数据异常。联系管理员，管理员会查看服务器日志排查问题，并尽力恢复数据。

游戏模式不对

如果发现游戏模式（生存、创造、冒险）不符合预期，先确认服务器当前设定的游戏模式。若与自己需求不符，礼貌地向我们提出调整请求，管理员会通过控制台或配置文件修改。

五、与其他玩家互动问题

  1. 遭遇恶意玩家：若遇到其他玩家破坏你的建筑、抢夺资源或语言攻击等恶意行为，不要与其对骂。立即截图保存证据，然后联系管理员，清晰说明事件经过和涉事玩家名称，管理员会依规处理。
  2. 想与他人合作：在游戏内聊天框积极发言，主动介绍自己和想法，寻找志同道合的小伙伴；也可以加入服务器官方交流群，在群里发布合作邀请，扩大交友范围。
